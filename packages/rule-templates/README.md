# rule-templates

模板化交易规则目录，负责定义规则模板、参数 schema、校验规则和标准化转换逻辑。

## 建议子目录

```text
src/
  templates/              规则模板定义
  normalizers/            表单输入到标准策略配置的转换
  validators/             参数合法性校验
```

## 使用原则

- 前端表单可以变化，但标准化规则结构必须稳定。
- 规则模板要足够简单，服务 MVP，不做复杂自由脚本。
- 每个模板都应有明确输入参数和生成结果。

## 当前阶段建议

优先支持：

1. 均线穿越
2. RSI 阈值
3. 连涨连跌
4. 突破新高新低
5. 盈利止盈与亏损止损
