# backtest-engine

回测引擎目录，负责将标准化策略配置应用到标准化历史数据上，输出稳定、可复现的回测结果。

## 建议子目录

```text
src/
  core/                   回测主流程与执行器
  metrics/                收益率、回撤、胜率等指标计算
  data/                   数据读取、数据适配、时间序列预处理
```

## 使用原则

- 只消费标准化输入，不依赖页面状态。
- 每次回测都应支持输入快照追溯。
- 指标计算口径必须固定。
- 同一输入重复运行时，结果必须一致。

## 当前阶段建议

优先完成：

1. 单标的
2. 单策略
3. 单周期
4. 只做多
5. 全仓进出
